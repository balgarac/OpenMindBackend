# talker

### 전체 상담원 리스트

### GET: /talker

> Require

```
none
```

> Response : Success

```
status - 200
data - {
	talkers = [
		{
			name - string
			phoneNumber - string
			age - number
			sex - string
			field - string
			area - string
			introduce - string
			profile - string
			token - string
			career - number
			openCount - number
		}
	]
}
```

### 태그에 의한 상담원 검색

### GET: /talkerBytag/:tag

> Require

```
tag - string
내가 선택한 태그
```

> Response: Success

```
status - 200
data = {
	talker = {
		name - string
		phoneNumber - string
		age - number
		sex - string
		field - string
		area - string
		introduce - string
		profile - string
		token - string
		career - number
		openCount - number
	}
}
```

> Response: Fail

```
status - 400
message - Bad request
tag 정보를 주지 않았을 때 발생.
```
